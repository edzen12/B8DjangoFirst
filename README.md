# Проект на Django Новостной сайт

#### для того чтобы скачать наш код, тебе нужен git (https://git-scm.com/)

#### открываем терминал и пишем туда
#### git clone https://github.com/edzen12/B8DjangoFirst.git
#### открываем скачанный проект через VSCode
#### в нем открываем терминал и пишем:
#### python -m venv venv
#### .\venv\Scripts\activate
#### pip install django pillow django-ckeditor-5 requests
#### python manage.py makemigrations
#### python manage.py migrate
#### python manage.py createsuperuser
#### python manage.py runserver
#### http://127.0.0.1:8000/
